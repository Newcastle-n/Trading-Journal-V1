# Trading Workspace

محیط کاری شخصی برای ژورنال، استراتژی‌ها و دانش معامله‌گری.

## اجرا (پیشنهادی)

دوبار کلیک روی:

- `start.bat` — سایت را باز می‌کند (نمونه قبلی را خودش می‌بندد)
- `stop.bat` — اگر لازم شد، سرور را کامل قطع می‌کند

با **بستن پنجره `start.bat`** یا `Ctrl+C` سرور هم خاموش می‌شود.

آدرس: `http://127.0.0.1:8765`

## اجرا از ترمینال

```bash
python server.py
```

قبل از استارت، اگر پورت اشغال باشد نمونه قبلی را خودش می‌بندد.

## میانبرها

- `Ctrl + K` — جستجو و Command Palette
- Sidebar — Dashboard / Journal / Knowledge / Backtests
- Settings — نام، اهداف٪، مسیر پوشه Screenshots/Videos

## داده

فایل‌های JSON در پوشه `data/` ذخیره می‌شوند. ژورنال و نکات از داخل سایت قابل ویرایش‌اند.
